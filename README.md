### <u>まず安全性！</u>
<BR><img src="images/4744b349-499b-49cf-a0fb-3a5e05590c6e.png" width="571" height="428"  /><BR><br>
この写真だけで大体伝わるのではないでしょうか<br>

まず第一に、**浅いので子供がおぼれる心配がありません**<br>
ほとんどのところは大人であれば膝下まで届かない程度には水深が浅く、溺れる心配はないです<br>

そして**川の周辺がめっちゃ整備されてます**<br>
ペンションから川への動線は芝生で整備されているので、ハブなどの脅威もほとんどないです<br>

ここまで安全なところはかなり珍しいです<br>

海みたいに岩が鋭くない、危険な生き物が少ないうえ、この好条件なので、<br>
沖縄で自然に触れられて、安全な場所として一番なんじゃないかな<br>
### <u>そして豊富な生態系！</u>
奥やんばるの里が位置するこの川は、<u>奥川の下流・ほぼ汽水域</u>にあたります<br>
そのため、満潮時には海水が入り込み、川と海の栄養が混ざり合う環境になっています<br>

その影響か、<br>
<span style="font-size:130%;">**整備された川としては珍しく、生き物がとても豊富**</span>です<br>
というか遊べるように整備された川自体が珍しい<br>
