# その魅力とは！！
## 奥川
このペンション最大の特徴<br>
なんと、**ペンションから約30mの距離に川があります**<br>

<span style="font-size:130%;">**沖縄の全ての川を巡った自分が断言します**</span><br>

<span style="font-size:150%;"><span style="color:#d70910;">**「家族で楽しめる川」としては圧倒的にNo.1です**</span></span><br>

<span style="font-size:150%;">**安全に遊べるよう整備されている**</span><br>

### <u>まず安全性！</u>
<BR><img src="images/4744b349-499b-49cf-a0fb-3a5e05590c6e.png" width="571" height="428"  /><BR><br>
この写真だけで大体伝わるのではないでしょうか<br>

まず第一に、**浅いので子供がおぼれる心配がありません**<br>
ほとんどのところは大人であれば膝下まで届かない程度には水深が浅く、溺れる心配はないです<br>

そして**川の周辺がめっちゃ整備されてます**<br>
ペンションから川への動線は芝生で整備されているので、ハブなどの脅威もほとんどないです<br>

ここまで安全なところはかなり珍しいです<br>

海みたいに岩が鋭くない、危険な生き物が少ないうえ、この好条件なので、<br>
沖縄で自然に触れられて、安全な場所として一番なんじゃないかな<br>
### <u>そして豊富な生態系！</u>
奥やんばるの里が位置するこの川は、<u>奥川の下流・ほぼ汽水域</u>にあたります<br>
そのため、満潮時には海水が入り込み、川と海の栄養が混ざり合う環境になっています<br>

その影響か、<br>
<span style="font-size:130%;">**整備された川としては珍しく、生き物がとても豊富**</span>です<br>
というか遊べるように整備された川自体が珍しい<br>

### <u>オススメのモリ</u>
結論から言うと、以下条件が揃ったものを買っておけばOKです<br>
- 3 ~ 4本刃
- ゴム付き

モリは「初心者なら3 ~ 4本刃、上級者なら1本刃」とよく聞きますが<br>
ウナギ相手であれば確実に3 ~ 4本刃がいいです<br>

ウナギは筋肉質で力が強く、そして思ったより肉が柔らかいので<br>
刺して5秒くらいでモリに絡みついて、自らの身体を切って逃走します<br>
<details><br>

一本刃はしっかり刺さるというメリットがありますが、三本刃でもウナギが柔らかいのでゴムのパワーだけで余裕で貫通します<br>
なので、当てやすく、身体を切りづらい3 ~ 4本刃が最適です<br>

ゴムはあるだけで狙いやすさが全然違うので出来ればあるやつを買いましょう<br>
> ※ゴム付きは禁止されている地域もあるので、使用する際はちゃんと調べましょう！場所によってはモリ事態×です<br>

釣り具屋にも結構置いてるのでそっちを探すなど、ただそれでも買える確証はないのでAmazonとかがいいかも<br>

### <u>いざ川へ</u>
モリとライト、クーラーボックス持っていざ川へ！<br>
ライトは自転車用の充電式ライトがオススメです　ちっちゃい割にめっちゃ強いです<br>
#### 探し方
ウナギは基本大きめの岩に纏うような形でいるので、大きめの岩を重点的に探しましょう<br>
近づいても基本逃げずにじっとしてるので、ゆっくり探して大丈夫です<br>
むしろ水面ギリギリまで顔近づけてよーく見ないと気づけないぐらい擬態能力が高いので一か所にしっかり時間かけましょう<br>
<BR><img src="images/3259a015-3703-4bcc-80de-78a842567398.jpg" width="494" height="659"  /><BR><br>
実際のだとこんな感じ<br>
自分の腕より太いサイズですら、間違えて踏んでから気づいた、なんてこともあるくらいにはわからないです<br>
こんなに近づいても1mmも動かないんですよね...<br>
#### 狙い方
基本頭が見えていることが多いので頭を狙いますが、突く瞬間にすごい勢いで後ろに下がるので、できる限り奥を狙いましょう<br>
あとできれば、逃げている最中に狙うのはやめた方がいいです<br>
割と当たりはするんですが、結局逃げられるので無駄に傷つけてしまいます<br>
#### 刺したあとは？
とにかくすぐに岸に向かいましょう<br>
ホントにすぐに外してきます<br>
あとは刺したときにできた傷口を手でつかんだり、口に手を突っ込んで噛まれに行くとしっかりホールドできます　かむ力は弱いのでそこまで危なくはないです　推奨はしませんが<br>
#### 閉じ込めるまで油断はするな！
岸に上がっても油断しないように、まだ勝負は終わってません<br>
ウナギは水陸両用、雨の日だと陸すら泳ぐ生き物です<br>
陸に降ろしてしまった時点でほぼ逃げられます<br>
籠とかに入れるまでは油断しないでください<br>
